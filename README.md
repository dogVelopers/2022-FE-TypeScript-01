# 2022 Dogvelopers FE - TypeScript

2022년도 개발자들 FE - 타입스크립트

## How to start

1. Fork this repo
2. Clone forked repo
3. Install dependencies

```bash
npm install
# or
yarn
```

4. Start project

```bash
npm run dev
# or
yarn dev
```
